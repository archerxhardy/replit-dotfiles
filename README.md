# Minimal Aesthetic Hyprland Dotfiles

Fast, smooth, and minimal configuration for Arch Linux.

## Features
- **Hyprland**: Smooth Bezier animations, rounded corners (12px), and heavy blur.
- **Waybar**: Dark transparent top bar with essential system info.
- **Kitty**: 85% transparent terminal with JetBrainsMono Nerd Font.
- **Rofi**: Aesthetic launcher and wallpaper selector.
- **Install Script**: Automated setup including drivers, audio, and bluetooth.

## Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd dotfiles
   ```

2. **Make the script executable:**
   ```bash
   chmod +x install.sh
   ```

3. **Run the installer:**
   ```bash
   ./install.sh
   ```

## Shortcuts
- `SUPER + Enter`: Open Kitty Terminal
- `SUPER + D`: App Launcher (Rofi)
- `SUPER + W`: Wallpaper Selector
- `SUPER + Q`: Kill Active Window
- `SUPER + H/J/K/L`: Move Focus
- `SUPER + SHIFT + H/J/K/L`: Swap Tiles
- `SUPER + 1-9`: Switch Workspace
